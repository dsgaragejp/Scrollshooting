# Stellar Vanguard - ゲームナラティブ設計書

## 概要

**ジャンル**: 横スクロールシューティング / スペースオペラ
**テーマ**: 銀河探査・惑星調査・艦隊戦闘
**インスピレーション**: スタートレック、銀河英雄伝説、グラディウス

---

## 世界観

### 設定

西暦2387年。人類は超光速航行技術「ワープドライブ」を完成させ、銀河系全域への探査が可能となった。

プレイヤーは「統合宇宙艦隊（United Stellar Fleet）」の戦闘機パイロットとして、探査艦隊「ヴァンガード」に所属。未知の星系を調査しながら、敵対勢力や宇宙の脅威から艦隊を守る任務に就く。

### 艦隊構成

```
ヴァンガード艦隊
├── USF ヴァンガード（母艦・探査旗艦）
├── 護衛艦 x 3
├── 補給艦 x 1
└── 戦闘機隊（プレイヤー所属）
    ├── アルファ小隊（プレイヤー）
    ├── ベータ小隊
    └── ガンマ小隊
```

## ビジュアルスタイル

### 背景グラフィック

XorDevスタイルの幻想的なシェーダーアートを採用：

- **星雲**: 流動的なノイズシェーダーによる幻想的な星雲表現
- **ワープ空間**: 歪んだ光線とレインボーグラデーション
- **惑星**: 大気散乱シェーダーによるリアルな惑星表現
- **宇宙塵**: パーティクルによる立体的な宇宙空間

### カラーパレット

| 環境 | 主要色 | アクセント |
|------|--------|-----------|
| 通常宇宙 | 深い紫・青 | 白い星々 |
| 星雲内部 | ピンク・オレンジ | シアンの光 |
| ワープ空間 | 虹色グラデーション | 白い光線 |
| 危険宙域 | 赤・オレンジ | 黄色の警告 |

## ストーリー構成

### プロローグ

統合宇宙艦隊は、銀河外縁部で謎の信号を検知。探査艦隊ヴァンガードが調査に向かう。

### Chapter 1: 出航

**ステージ1-1: 訓練航海**
- 基本操作チュートリアル
- 小規模な隕石群を突破

**ステージ1-2: 初遭遇**
- 未知の敵機との初戦闘
- 敵の正体は不明

### Chapter 2: 星雲探査

**ステージ2-1: オリオン星雲突入**
- 幻想的な星雲内部を航行
- 視界不良の中での戦闘

**ステージ2-2: 星雲の番人（ボス）**
- 巨大な生体兵器との戦闘

### Chapter 3: 惑星調査

**ステージ3-1: 惑星アルカディア軌道**
- 惑星周回軌道での戦闘
- 背景に美しい惑星

**ステージ3-2: 大気圏突入**
- 下方向スクロールへ変化
- 大気圏突入エフェクト

**ステージ3-3: 地表調査**
- 惑星地表での戦闘
- 古代遺跡の発見

### Chapter 4: 艦隊危機

**ステージ4-1: 隕石群襲来**
- 巨大隕石群から艦隊を守る
- **母艦撤退イベント**発生

**ステージ4-2: 殿軍戦**
- 艦隊が撤退する間、敵を足止め
- タイムリミット付きサバイバル

### Chapter 5: 真実

**ステージ5-1: 敵本拠地潜入**
- 敵の正体が判明
- ステルスミッション要素

**ステージ5-2: 最終決戦（ボス）**
- 敵旗艦との決戦

### エピローグ

銀河の平和を取り戻し、次なる探査へ旅立つヴァンガード艦隊。

## 特殊イベント

### 1. 母艦撤退イベント

```
発生条件: ステージ中盤でトリガー
演出:
1. 警報音「全艦隊に告ぐ。緊急ワープ準備」
2. 背景で母艦がワープ準備開始
3. プレイヤーは敵を食い止める
4. タイムリミット後、母艦ワープ完了
5. プレイヤーも追従ワープ
```

### 2. 護衛任務イベント

```
発生条件: 特定ステージ
演出:
1. 補給艦を護衛しながら航行
2. 補給艦のHPゲージ表示
3. 補給艦破壊でミッション失敗
```

### 3. 艦隊合流イベント

```
発生条件: ボス戦前
演出:
1. 援軍艦隊がワープアウト
2. 一時的に火力増強
3. 艦隊戦の迫力ある演出
```

### 4. 惑星軌道変化イベント

```
発生条件: 惑星ステージ
演出:
1. 惑星の重力で軌道変化
2. スクロール方向が変化
3. 惑星大気圏への突入
```

## 敵勢力

### メカノイド（機械生命体）

銀河外縁部から侵入してきた謎の機械生命体。

| タイプ | 説明 |
|--------|------|
| ドローン | 小型偵察機、大群で出現 |
| ファイター | 中型戦闘機、編隊行動 |
| クルーザー | 大型巡洋艦、砲台多数 |
| キャリアー | 空母、ドローンを射出 |
| コア | ボス、巨大コアユニット |

### 宇宙生物

星雲内部に生息する巨大生物。

| タイプ | 説明 |
|--------|------|
| スターワーム | 長大な宇宙蛇 |
| ネビュラジェリー | 星雲クラゲ、電撃攻撃 |
| コスモドラゴン | 伝説の宇宙竜（隠しボス） |
