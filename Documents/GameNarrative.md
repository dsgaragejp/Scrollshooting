# Stellar Vanguard - ゲームナラティブ設計書

## 概要

**ジャンル**: 横スクロールシューティング / スペースオペラ
**テーマ**: 銀河探査・惑星調査・艦隊戦闘・時空を超えた絆
**インスピレーション**: インターステラー、スタートレック、グラディウス

---

## 世界観

### 設定

西暦2387年。地球は環境破壊により居住不可能になりつつあった。

人類最後の希望として、土星軌道上に出現した「ワームホール」を通じて新天地を探す「ラザロ計画」が発動。12の探査船が送り出されたが、通信途絶。

プレイヤーは探査艦隊「エンデュランス」の戦闘機パイロット。ワームホールの向こう側で先遣隊の痕跡を追い、人類の新たな故郷となる惑星を探す任務に就く。

### テーマ

> *"Love is the one thing that transcends time and space."*
> 「愛は時間と空間を超越する唯一のもの」

プレイヤーは地球に残した家族への想いを胸に、未知の銀河を探査する。

### 艦隊構成

```
エンデュランス艦隊
├── USF エンデュランス（母艦・回転リング型探査船）
│   ├── 居住モジュール x 4（回転による人工重力）
│   ├── ランダー（惑星着陸船）x 2
│   └── レンジャー（高機動探査機）x 2
├── 護衛艦 x 2
└── 戦闘機隊（プレイヤー所属）
    ├── レンジャー小隊（プレイヤー機）
    └── ランダー小隊（重装）

AI支援システム:
├── KIPP（艦隊管制AI）
├── TARS（戦術支援AI・プレイヤー僚機）
└── CASE（緊急対応AI）
```

## インターステラー要素

### 1. ブラックホール「ガルガンチュア」

巨大ブラックホールが背景に登場。降着円盤と重力レンズ効果を再現。

```
視覚効果:
- 光の歪み（グラビテーショナルレンズ）
- 降着円盤（オレンジ〜白のグラデーション）
- 事象の地平線（完全な黒）
- 相対論的ジェット
```

**ゲームプレイへの影響:**
- ブラックホール接近時、弾道が曲がる
- 時間の流れが遅くなる演出
- 吸い込まれると即死ゾーン

### 2. ワームホール航行

ステージ間の移動でワームホールを通過。

```
演出:
1. 球体状のワームホール出現
2. 突入時に空間が歪む
3. トンネル状の超空間
4. 出口で新しい星系へ
```

**シェーダー効果:**
- 球面上に映る向こう側の宇宙
- 通過時のレインボー歪み
- 時空のトンネルエフェクト

### 3. 時間遅延（タイムダイレーション）

ブラックホール接近ミッションでは時間の流れが変化。

```
演出例:
- ミッション中: 「現地時間: 1時間経過」
- ミッション後: 「地球時間: 23年経過」
- 地球からの古いメッセージ受信
```

**感情的インパクト:**
- 家族からの成長した姿のビデオメッセージ
- 時を超えた別れと再会

### 4. ミラーの惑星（水の惑星）

巨大な波が押し寄せる海洋惑星ステージ。

```
ステージギミック:
- 背景に巨大な波（高さ1km級）
- 波に飲まれると大ダメージ
- タイミングを見て進行
- 時間制限あり（タイムダイレーション）
```

### 5. マンの惑星（氷の惑星）

凍った雲が浮かぶ極寒の惑星。

```
ステージギミック:
- 氷の雲（障害物）
- 裏切りイベント（NPCが敵に）
- 極寒エフェクト（視界制限）
```

### 6. テセラクト（五次元空間）

クライマックスで五次元空間に突入。

```
演出:
- 無限に続く本棚のような構造
- 過去の自分との通信
- 時間を超えたパズル要素
- 重力を使った信号送信
```

### 7. ドッキングシーケンス

回転する母艦への緊急ドッキング。

```
特殊ステージ:
- 母艦が制御不能で回転
- 回転に合わせてドッキング
- 「それは不可能だ」「いや、必要なんだ」
- 成功時の達成感
```

## ストーリー構成

### プロローグ

地球の荒廃。ワームホールの発見。ラザロ計画の発動。
プレイヤーは家族（娘）との別れを経て、エンデュランスに乗り込む。

> 「必ず戻る。約束だ」

### Chapter 1: 出航

**ステージ1-1: 土星軌道**
- チュートリアル
- 土星のリングを背景に訓練飛行

**ステージ1-2: ワームホール突入**
- ワームホールへの接近
- 初めての超空間航行
- TARS「正直度設定: 90%」

### Chapter 2: 新銀河

**ステージ2-1: 未知の星系**
- ワームホールの向こう側
- ガルガンチュアが遠景に見える
- 敵勢力との初遭遇

**ステージ2-2: 先遣隊の信号**
- 12年前に送られた探査船の痕跡
- 信号を追って惑星へ

### Chapter 3: ミラーの惑星（水の惑星）

**ステージ3-1: 軌道降下**
- 惑星への接近
- 時間遅延の警告

**ステージ3-2: 巨大波**
- 海洋惑星での戦闘
- 巨大な波を避けながら進行
- **時間制限イベント**（1時間 = 地球7年）

**ステージ3-3: 緊急脱出**
- 波に追われながら離脱
- 艦隊との合流

*ミッション後: 23年分のビデオメッセージ受信*

### Chapter 4: マンの惑星（氷の惑星）

**ステージ4-1: 氷の雲海**
- 凍った雲の中を航行
- 先遣隊員マンの基地発見

**ステージ4-2: 裏切り**
- マンの裏切りイベント
- 母艦へのテロ攻撃
- **緊急ドッキングイベント**（回転する母艦へ）

### Chapter 5: ブラックホール

**ステージ5-1: ガルガンチュア接近**
- ブラックホール周辺での戦闘
- 弾道が曲がる特殊ステージ
- 降着円盤の美しさと危険

**ステージ5-2: 事象の地平線**
- ブラックホールへの突入
- 時間が無限に引き伸ばされる演出

### Chapter 6: テセラクト

**ステージ6-1: 五次元空間**
- 時空を超えた空間
- 過去との通信パズル
- 「STAY」のモールス信号

**ステージ6-2: メッセージ**
- 重力を通じて過去へ信号送信
- ブラックホールの特異点データ送信

### エピローグ

ワームホールを通じて土星軌道へ帰還。
しかし、プレイヤーにとっては数時間、地球では100年以上が経過。
コロニーで老いた娘との再会。

> 「約束通り、戻ってきたよ」

## 特殊イベント

### 1. タイムダイレーション（時間遅延）

```
発生条件: ブラックホール接近ステージ
演出:
1. 画面端に「現地時間」「地球時間」表示
2. ミッション進行で地球時間が加速
3. ミッション後、蓄積されたメッセージ受信
4. 家族の成長・老いを描写
```

### 2. ビデオメッセージ受信

```
発生条件: 時間遅延後
演出:
1. 母艦のモニターに映像
2. 家族からの数十年分のメッセージ
3. 成長する子供、老いる親
4. プレイヤーの感情を揺さぶる
```

### 3. 緊急ドッキング

```
発生条件: Chapter 4ボス後
演出:
1. 母艦が爆発で回転開始
2. 「ドッキング不可能です」(KIPP)
3. 「不可能じゃない。必要なんだ」
4. 回転に合わせてミニゲーム
5. 成功でカタルシス
```

### 4. 母艦撤退イベント

```
発生条件: ステージ中盤でトリガー
演出:
1. 警報「全艦、緊急ワープ準備」
2. 母艦がワームホール生成開始
3. プレイヤーは敵を足止め
4. タイムリミット後、母艦脱出
5. プレイヤーも追従
```

### 5. テセラクト通信

```
発生条件: Chapter 6
演出:
1. 五次元空間で過去を見る
2. 重力で本棚を動かす
3. モールス信号でメッセージ
4. 「愛は時空を超える」
```

## AI僚機システム

### TARS（戦術支援AI）

プレイヤーの僚機として常に随伴。

```
特徴:
- 独特のユーモア設定（正直度90%）
- 戦闘中のアドバイス
- 危機的状況での自己犠牲
- 「これは不可能だ」「確率は低いが、やるしかない」
```

### 会話例

```
TARS: 「正直度を90%に設定しています」
プレイヤー: 「なぜ100%じゃない？」
TARS: 「100%の正直さは...関係を壊すことがあります」

TARS: 「ドッキング成功確率: 0.3%」
プレイヤー: 「TARS、確率を表示するな」
TARS: 「了解。確率非表示モード」
```
